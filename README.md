# SR-mapping

[![DOI](https://zenodo.org/badge/543010687.svg)](https://zenodo.org/badge/latestdoi/543010687)

Code for van den Brink, RL, Hagena, K, WIlming, N, Murphy, PR, Büchel, C, & Donner, TH. (2022). Flexible sensory-motor mapping rules manifest in correlated variability of stimulus and action codes across the brain. Neuron.
 
This repository contains matlab code to:
- Reproduce the main figures and statistical results.
- Run informational connectivity analyses (see folder simulations/functions)
